## 1. 产品概述

一款纯浏览器端的在线 Markdown 编辑器，提供实时预览、文档大纲、主题切换、PDF 导出等完整功能，无需任何服务器支持即可使用。

- 主要用途：为开发者、写作者提供轻量、高效的 Markdown 写作与预览工具
- 核心价值：零依赖、开箱即用、功能完备的 Markdown 编辑体验

## 2. 核心功能

### 2.1 功能模块

1. **主编辑页面**：
   - 左侧 Markdown 输入编辑区
   - 右侧实时 HTML 渲染预览区
   - 左侧文档大纲导航栏
   - 顶部工具栏（主题切换、导出、复制等操作）

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主编辑页 | 编辑区 | 支持 Markdown 语法输入，带行号，支持输入同步滚动 |
| 主编辑页 | 预览区 | 实时渲染 HTML，代码块语法高亮，与编辑区同步滚动 |
| 主编辑页 | 文档大纲 | 根据标题层级（H1-H6）自动生成导航树，点击跳转对应位置 |
| 主编辑页 | 主题切换 | 亮色/暗色主题切换，配色通过 CSS 自定义属性统一控制 |
| 主编辑页 | PDF 导出 | 将渲染后的内容导出为 PDF 文件 |
| 主编辑页 | HTML 复制 | 一键复制渲染后的 HTML 片段到剪贴板 |

## 3. 核心流程

用户打开页面后，可直接在左侧编辑区输入 Markdown 内容，右侧实时预览渲染效果。通过左侧大纲快速导航文档结构，可随时切换主题、导出 PDF 或复制 HTML。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：采用编辑工具风格，使用深灰/深蓝作为主色，辅以蓝绿色强调色
- **亮色主题**：米白背景 + 深灰文字 + 蓝绿色强调
- **暗色主题**：深蓝灰背景 + 浅灰文字 + 蓝绿色强调
- **字体**：等宽字体用于编辑区和代码块，优雅的无衬线字体用于正文
- **布局**：三栏布局（大纲 | 编辑区 | 预览区），顶部工具栏
- **按钮风格**：圆角、图标+文字、悬停过渡动效

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 主编辑页 | 工具栏 | 主题切换按钮、导出 PDF 按钮、复制 HTML 按钮，带图标 |
| 主编辑页 | 大纲栏 | 树形层级缩进，当前标题高亮，悬浮效果 |
| 主编辑页 | 编辑区 | 等宽字体、行号、等宽间距、焦点高亮 |
| 主编辑页 | 预览区 | 排版优美、代码块带语言标签、表格美化 |

### 4.3 响应式

桌面端优先设计：三栏并列布局；在窄屏下可折叠大纲栏，编辑区与预览区上下排列。
